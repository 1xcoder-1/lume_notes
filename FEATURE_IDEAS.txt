# Chat with PDF

The Chat with PDF feature allows you to upload existing documents into your note and use the AI Assistant to discuss the PDF's content.

## How it works

Inside any note, you can click the "Chat / Import PDF" button. This uploads your file directly to our secure server, and the AI Assistant reads the document to understand it.

## Key Features

1. **Instant PDF Import**: Strictly import any PDF file to add its text directly into your current note. Other formats like images or Word documents are currently unsupported to ensure parsing quality.
2. **AI Question & Answer**: Ask the AI questions about the PDF (e.g., "Summarize this PDF" or "What are the main points?").
3. **Multi-Document Support**: Chat with multiple PDFs to find connections between them.

## Technical Details

- **Location**: `apps/app/notes/page.tsx` (handlePDFUpload) and `apps/web/components/sidebar-content.tsx` (Upload UI).
- **Enforcement**: We use client-side validation to ensure only `.pdf` files are accepted, preventing accidental uploads of incompatible formats.
- **Processing**: High-quality text extraction from PDF pages, supporting complex layouts and fonts.
